Here are some places where I have worked.

Quality Assurance in Big Business

My first summer internship found me at IBM in Costa Mesa, CA. There, I worked on implementing Python and Java based automation tools to configure and install IBM Case Manager on Linux, AIX, and Windows Server systems. I also automated the use of an existing JavaScript testing suite.

Solving Problems in Tech Support

While back at Olin, I work in the IT department where I diagnose, troubleshoot, and repair technical problems on student and faculty computers. This includes replacing PC components, recovering personal data, and interfacing with staff and students to figure out technical issues on campus.

Keeping a Watchful Eye on Patrons

My first job was as lifeguard at the local city pool. There, I assured the safety of patrons and assisted those in need during the summer. This involved monitoring patrons, keeping team members updated on issues, reporting to superiors on problems, and maintaining the facilities for a large pool.

Here are some of the projects I have worked on.

Whether it be developing sentiment analysis software for social networks, coding a Python game rendered exclusively in OpenGL, or building a mechanized beaver that swims in water, this first year at Olin has been full of projects.

Space Team Down

Worked with a team of five to create a Python survival game, rendered in OpenGL. Resource collection and enemy avoidance were key to survival.

Sentiment Analysis

Worked with another student to create a Python program which analyzed the possible success of Facebook posts, depending on previous user data.

Mechanized Beaver

Worked in a team to design and build a mechanized beaver toy. It was able to swim in a pond and pick up logs, which were used to save a pup.

Modeling Dialysis

Worked with another student to model the rate of dialysis. This involved using real world data and a calibrated theoretical dialysis model using MatLab.

Impossible Game

Worked with another student to develop a GUI based survival game, where the player had to avoid randomly generated objects for as long as possible.

Following Robot

Worked with a partner to create a line following robot. The use of photodiodes and servos guided the robot to follow a black line on the ground.
